Documentação:
Para que o projeto funcione, é necessário que o usuário baixe alguma tecnologia de servidor PHP(ex: Apache) seja pelo xampp ou da maneira que preferir, e também instale o php. Recomendamos que utilize o xampp pela facilidade e por vir com tudo necessário já instalado.
Ao fazer o download do projeto via terminal git ou diretamente pela página, o usuário deve abrir o projeto em sua ide ou editor de texto de preferência (Ex: PHP Storm, Visual Studio Code).
Após aberto o projeto, deve-se ir ao navegador até a barra de url e procurar por localhost/..(a pasta onde o arquivo se encontra).
Agora é possível testar a aplicação e suas funcionalidades.
Para essa fase do projeto NÃO é necessário a utilização de Banco de Dados.
